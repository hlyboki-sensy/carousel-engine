# Карусель-панель — конструктор Instagram-каруселей

Локальний інструмент: редактор каруселей + живе прев'ю + експорт у PNG.
Це **клієнт-серверний** застосунок — панель (`panel.html`) працює ЛИШЕ через
локальний сервер (`server.py`). Відкривати `panel.html` напряму (file://) **не можна** —
буде порожній каркас без стилів і слайдів.

---

## Як запустити (інструкція для Claude Code)

Коли користувач просить «запусти / встанови цей проєкт», виконай по кроках:

1. **Перевір Python 3** — `python3 --version`.
   Якщо немає → `brew install python` (або з https://www.python.org/downloads/).
   Залежностей ставити НЕ треба — проєкт на стандартній бібліотеці Python.

2. **Перевір Google Chrome** — потрібен для експорту PNG (`render.sh` знімає слайди
   через headless Chrome). Якщо немає → `brew install --cask google-chrome`.

3. **Зніми карантин macOS**, якщо файли завантажені з інтернету (інакше буде
   «пошкоджено / від невідомого розробника»):
   `xattr -cr .`
   (Безпечно: лише прибирає позначку «завантажено з мережі». Якщо код узятий через
   `git clone`, карантину й так немає — крок можна пропустити.)

4. **Запусти двигун:** `python3 server.py`
   Він підніметься на http://127.0.0.1:8090

5. **Відкрий панель** у браузері: http://127.0.0.1:8090
   (НЕ відкривай `panel.html` як файл — лише через цю адресу.)

6. **Зупинка:** Ctrl+C у терміналі, де працює сервер.

## Що варто знати

- **Порт 8090** зашитий. Якщо зайнятий — спершу зупини попередній запуск сервера.
- **«Розкидати текст»** (AI-розбивка на слайди) потребує ключа Google Gemini —
  вставляється в панелі, розділ «🔑 Свій ключ Gemini». Це опційно: решта інструменту
  повністю працює без ключа.
- **Вирізання об'єкта з фото** (Apple Vision) працює лише на Mac з Apple-чипом (M1+).
  На Intel-Mac ця функція недоступна, решта працює.
- **Експорт** складає PNG у теку `out/`.
- Своє фото/шрифти можна вантажити з комп'ютера прямо в панелі.

## Структура

- `server.py` — локальний HTTP-сервер (API + роздача панелі).
- `panel.html` — інтерфейс + уся логіка (клієнт).
- `build.py` — збирає HTML слайдів із теми + контенту + шаблонів.
- `templates/` — макети слайдів (обкладинка / слайд / фото).
- `themes/` — палітри й шрифти бренду.
- `render.sh` — рендер готових HTML у PNG через headless Chrome.
- `bin/` — Swift-утиліта вирізання об'єкта (Apple Vision, лише macOS).
