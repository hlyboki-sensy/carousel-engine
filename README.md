# Carousel Engine

Движок каруселей для Instagram (3:4, 1080×1440). Слайди збираються з трьох
**незалежних** частин — тому «різні шрифти, різні кольори» = просто інша тема,
а не переверстка кожного слайда.

```
тема (як виглядає)  +  контент (що написано)  +  шаблон (структура)  →  PNG
```

## Структура

```
carousel-engine/
  themes/          пресети вигляду (шрифти, кольори, кікер, хендл)
    hlyboki.json     — @hlyboki_sensy: Didot + Helvetica, золото #FFD20E
    noir.json        — контрастний: Avenir Next + теракота #E2533B
  templates/       макети на CSS-змінних (структуру не чіпаємо)
    cover.html       — обкладинка: фото + кікер + заголовок (2 рядки) + підзаголовок
    text.html        — текстовий слайд: світлий фон, теза + абзац
    photo.html       — фото-слайд: фон-фото + градієнт + теза + текст
  content/         сценарії каруселей
    example.json     — приклад (карусель «Час»)
  build.py         збирає out/NN.html з теми + контенту
  render.sh        рендерить out/NN.html → out/NN.png (headless Chrome)
  out/             результат
```

## Запуск

```bash
python3 build.py                                  # тема + контент за замовчуванням
python3 build.py themes/noir.json content/x.json  # своя тема + свій контент
bash render.sh                                    # HTML → PNG
```

## Як описати карусель (content/*.json)

Кожен слайд — об'єкт із `type` (`cover` | `text` | `photo`) і полями тексту:

```json
{
  "slides": [
    { "type": "cover", "photo": "фото/cover.jpg",
      "ЗАГ_1": "«котра", "ЗАГ_2": "година?»",
      "ПІДЗАГ": "дитина не&nbsp;питає — вона просто живе" },
    { "type": "text", "ТЕЗА": "Лічба", "ТЕКСТ": "Десь ми навчилися рахувати…" },
    { "type": "photo", "photo": "фото/далеко.jpg",
      "ТЕЗА": "Не&nbsp;прожитий", "ТЕКСТ": "Рік минув…" }
  ]
}
```

Шляхи до фото — відносно файлу контенту.

**Типографіка:** короткі слова (1–2 літери) клей до наступного через `&nbsp;`,
щоб не висіли в кінці рядка: `не&nbsp;питає`, `від&nbsp;нас`.

## Додати новий бренд / клієнта

Скопіюй `themes/hlyboki.json`, зміни шрифти й кольори, збережи як
`themes/{бренд}.json`. Контент і шаблони лишаються ті самі.
