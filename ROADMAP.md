# Карусель-панель — доопрацювання (замовлення Олени, 2026-07-02)

«Ресурс» = веб-панель `panel.html` + `server.py` + `build.py` + `templates/` + `themes/`.
Запуск: `python3 server.py` → http://127.0.0.1:8090

## 7 вимог і рішення

1. **Шрифти брендів** — РІШЕННЯ: поки лишаємо системні (Didot/Bodoni з macOS). Файли шрифтів не зашиваємо.
2. **Друга фотографія на слайд** — додано верхній шар (вирізаний об'єкт / друге фото) поверх тексту.
3. **Завантажити текст → авто-розкидання по слайдах** — РІШЕННЯ: автоматично через Opus 4.8.
   Ключ `ANTHROPIC_API_KEY` читається з `../.env`. Ендпоінт `/api/split-text`.
4. **Елементи дизайну**: кікер «роздуми маркетолога» (є), вставлення тексту (є),
   + **вирізати об'єкт як в iPhone** → рідний Apple Vision (`bin/cutout`, Swift). Об'єкт лягає
   верхнім шаром над текстом. Компіляція: `bash bin/build-cutout.sh` (проти SDK 15.5, бо
   дефолтний SDK 26.2 не збирається з наявним swift 6.1.2).
5. **Однотонне тло за hex** — на слайді перемикач «фото / колір» + вибір коду кольору.
6. **Біле тло панелі** — інтерфейс переведено на світлу тему.
7. **Кроп + масштаб фото + формат каруселі** — контроли зсуву(X/Y)+масштабу; формати 3:4, 4:5, 1:1.

## Контракт даних слайда (JSON)
```
type: cover|text|photo
ЗАГ_1,ЗАГ_2,ПІДЗАГ (cover) | ТЕЗА,ТЕКСТ (text/photo)
bgMode: photo|color         # чим заповнено тло
bgColor: "#RRGGBB"
photo: "шлях"
posX,posY: 0..100           # кроп фону (object-position %)
scale: 1.0..3.0             # зум фону
cutout: "шлях.png"          # вирізаний об'єкт (верхній шар, над текстом)
cutX,cutY: 0..100 · cutScale: 0.3..2.0
```
Формат каруселі — глобальний (в overrides): 3:4=1080×1440 · 4:5=1080×1350 · 1:1=1080×1080

## Технічне середовище (важливо)
- Python 3.14 (системний). `.venv` ЗЛАМАНИЙ (шлях до перейменування теки) — не використовувати.
- Панель і build — лише stdlib, .venv не потрібен.
- rembg/onnxruntime НЕ ставляться на 3.14 → cutout робимо через Swift/Vision.
- Swift 6.1.2 + SDK 15.5. Chrome headless для рендера PNG (render.sh).
